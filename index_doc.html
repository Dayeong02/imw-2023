Design Elements

1. Section-wise Scroll and Navigation Bar: Each section contains the main content of the page, and scrolling moves smoothly to the next section. Additionally, there are navigation bars at the top, left, bottom, and right, allowing users to scroll to the corresponding sections by clicking on the menu.

2. Text Hover and Click: The text in each section changes colour when hovered over, and clicking on it takes the user to the relevant link.

3. Go to Top Button: A "Go to Top" button is positioned at the bottom right of the page, enabling users to instantly return to the top of the page. The area around the button is highlighted with a dashed line effect, adding animation to the website.

4. Mouse Cursor: The user's mouse cursor is designed as a transparent circle. When hovering over clickable elements, the cursor enlarges to emphasize those elements.

5. Gradient Background: The page background features a gradient colour effect, providing a smooth transition between colours and preventing the page from appearing monotonous.

Functionalities

Typing Effect: Text in each section displays a typing effect as the user scrolls, gradually revealing the characters.

Fade-in Effect: List items 'li' fade in smoothly as the user scrolls, creating a subtle visual effect.

Keyboard Interaction: Pressing any key generates randomly selected emojis on the screen. These emojis disappear after a certain period.

Furthermore, the website integrates responsive design, guaranteeing a seamless adjustment of the layout to diverse screen sizes while preserving content integrity. It ensures that users, whether on desktops, tablets, or mobile phones, encounter a visually captivating and well-structured interface devoid of any visual distortions or content misalignments.
