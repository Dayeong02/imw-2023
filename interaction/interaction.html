Keyboard Symphony

Start page

In the centre of the screen, there's a virtual computer monitor displaying the text "Press a key to play a sound and see visuals!". The design of this page is reminiscent of a traditional computer recording studio. The monitor and keyboard follow the design of a traditional computer, and the background colour of the screen is set to a neutral gray, allowing users to focus on the audio and visual effects.


The page is implemented using HTML, CSS, and JavaScript, utilizing the Web Audio API to generate sounds based on keyboard inputs. Additionally, CSS animations are used to visually represent the musical notes.

After-Keystroke Effects

When users press keys on their keyboard, they hear musical tones and see random musical notes appear on the screen.

"Keyboard Symphony" is an engaging web application that allows users to unleash their creativity and experience musical enjoyment. It provides users with a unique experience of both music and visual effects, making it a fun and entertaining experience.
